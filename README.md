# Guess the Number Game

A simple web-based number guessing game built with Node.js, Express.js, and EJS.

## Features

- Guess a random number between 1 and 100
- Track the number of attempts
- Leaderboard to display top scores
- Clean and responsive UI

## Technologies Used

- Node.js
- Express.js
- EJS (Embedded JavaScript Templates)
- HTML/CSS

## Installation

1. Clone the repository
2. Run `npm install` to install dependencies
3. Run `npm start` to start the server
4. Open `http://localhost:3000` in your browser

## How to Play

1. Click "Start Game" on the home page
2. Enter your name (optional) and make a guess
3. The app will tell you if your guess is too high or too low
4. Keep guessing until you get it right
5. Your score will be added to the leaderboard

## Project Structure

- `server.js` - Main server file
- `views/` - EJS templates
- `public/` - Static files (CSS)
- `package.json` - Project dependencies and scripts

This project demonstrates basic web development skills with Node.js, including routing, templating, and form handling.
